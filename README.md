# Nexus Dev Financial System (Система обліку та аналізу фінансів)

Програмний комплекс призначений для автоматизації процесів обліку, пагінації та аналізу транзакцій на різноманітні внутрішні потреби компанії. Проєкт розроблено в межах лабораторної роботи з дисципліни "Програмування, алгоритмізація та основи промпт-інжинірингу".

## 🚀 Основний функціонал

* **Управління транзакціями:** Підтримка різних типізацій фінансових операцій (`Income` — Дохід, `Expense` — Витрата) з використанням перелічуваних категорій (`Category`).
* **Поліморфна ієрархія:** Розрахунок кінцевого балансу залежно від типу транзакції з перевизначенням магічних методів `__str__` та операторів додавання `__add__` / `__radd__`.
* **Контекстне керування та ліміти:** Автоматична перевірка балансу гаманця перед проведенням витрат, що унеможливлює утворення від'ємного балансу (овердрафту).
* **Просунута обробка даних:** Використання функторів (`FinanceAnalytics`) для динамічного обрахунку комісійних зборів на витрати (1.5%) та власних ітераторів/генераторів (`generate_statement_pages`) для посторінкового виведення звіту (пагінації).
* **Персистентність даних:** Повне збереження та відновлення стану системи за допомогою бінарної серіалізації (`pickle`), читання зовнішніх налаштувань із конфігураційного файлу `config.json` та експорт фінансових звітів у формат `csv`.
* **Інтерактивний CLI-інтерфейс:** Зручне текстове меню на базі оператора `match/case` з високим рівнем стійкості до некоректного введення даних користувачем (`Fault Tolerance`).

## 🛠️ Вимоги до середовища

Для запуску проєкту та збирання пакунка необхідне наступне програмне забезпечення:
* Python версії 3.10 або новішої.
* Інструмент збирання `setuptools` (вбудований у сучасні версії Python) або менеджер пакунків `pip`.
